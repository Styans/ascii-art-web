## ASCII-ART-FS
- The usage must respect this format go run ./cmd/app/asciiArtTerminal/main.go [STRING] [BANNER], any other formats must return the following usage message:
```
Usage: go run ./cmd/app/asciiArtTerminal/main.go [STRING] [BANNER]

EX: go run ./cmd/app/asciiArtTerminal/main.go something standard
```
